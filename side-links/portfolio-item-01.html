Add to Cart Mobile App

Mobile App

This project is a web app which can be added to your mobile device to be a mobile app. The project was build using HTML, CSS, JavaScript and Firebase.

The function of the app is to be able to add items you need to buy when you go out to shop. You simply add the items inside the input field and press the add button which will add the item below inside a list viewed as a box. The items can be deleted by double clicking the item, which will delete it from the interface and the database.

I used HTML to create the markup of the content/project, used CSS to add the style and hover effects. Then i used the Javascript for DOM manipulation in order to be able to add new lists items inside the unordered list inside the DOM, I also used the Javascript to add eventlisteners for click actions for the add button and for the deletion of the items and also used javascript in order to access the firebase for using the database to store the items from different users and also to make sure that the items don't get removed when the app gets closed and re-opened.
