CodeSpace Pets Instagram Page

Instagram page project

This project is a clone of a instagram page, it resembles how an instagram page looks and also has responsive design for different screen sizes. The project utilises HTML and Javascript

The function of this project is to showcase expertise in responsive design, grid layouts and flex layouts. It also includes features of adding hover effects that change the look of images when hovered over and resembling instagram by adding the likes and comments icons inside the images when hovered over.

I used HTML to mark up the content of page. Used the CSS to add style to the page, added grid layout for the gallery section to have a three by three grid, also added grid to the profile section to structure the profile to resemble instagram. Inside the profile grid I utilised flex to structure the content properly. I also utilised media queries to make the page responsive when moving to different screen sizes, also changed the layout style when using media queries.
