Fifa TournamentPlayer counter App

Player counter App

This project is a simple player counter app, which you can use to count players for each session and save that number. The project utilises HTML, CSS and Javascript.

The function of this project is to add the number of player per fifa tournament sessions, for each tournament that runs you can add the number of player who entered for that specific session and that can be saved and displayed on the interface. There is an increment button which when click it increases the number of players and there is a save button which when clicked saves the number and creates a new set from zero to add a new number of players.

The project utilises HTML for the markup of the project. Also uses CSS for the styling. Then Javascript is used to get access to the DOM and able to functions for increment and save function. The functions are called on the html for onclick events.
