Black Jack Mini Game

Black Jack project

This project is a mini game of black jack which provides card numbers when you start the game and has options to get given more card numbers to get a chance to win. Has conditions for win or loss. The project utilises HTML, CSS and Javascript.

The function of this project is to showcase HTML, CSS and Javascript skills. It has a start button to provide numbers to resemble cards and if the sum of the card numbers is below 21 you have the option to get a new card number by pressing the button new card. If sum is under 21 you can still draw more card but if over 21 you loss and if equal to 21 you win.

I used HTML for markup of the project, used CSS for styling and hovering effect. Then used Javascript to get access to the DOM to be able to add new content to the DOM and to be able to add eventlisteners for the buttons for clicks. Also added conditions for when you need a new card or lose or win. Also utilised a loop to get access to arrays and display the numbers of cards which are stored in the array. I also have a funtion which has the maths object which is used to get random numbers which are used for the first and second card number you get from the array and also used on the function/button for getting a new card.
