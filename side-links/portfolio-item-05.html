Code Pen Custom Buttons

CSS buttons

This project is a CSS project to test different styles of animations and transitions using a platform called Code pen.

The function of this project is to test my skills in CSS by applying different kinds of styling on buttons and using hover and focus having different transitions and animations using keyframes.

For this project I only used HTML and CSS. The HTML is used for markup adding three buttons. The CSS is used for styling and adding hovering effects which have transitions and animations. For the animations I had to use keyframes which add different colors with effects for the time function, effect and duration of the animation. I also added shadows for the buttons for a neon look.
